# 版权接口演示平台-剧本创作平台

## Dependencies
- lucide-react: 图标库，用于UI图标展示

## Architecture
- 路由系统：TanStack Router，页面位于 src/routes/
- 组件结构：通用组件在 src/components/，类型定义在 src/types/
- **双层架构**：`/` 为演示平台登录入口 → `/platform` 选择演示项目 → `/app/*` 进入具体平台（如剧本创作）
- 协议见证平台（电商类）：首页(/app/agreement)、登录注册(/app/agreement/login)、个人中心(/app/agreement/profile)；协议详情弹窗采用PDF文档布局：左上角TSA LOGO、中间标题居中、右上角账户信息+确认时间+版本+认证标识；首页和个人中心顶部导航栏左侧显示"返回平台"按钮，点击返回/platform平台选择页；首页标题为"格式协议见证服务演示平台"；通知公告功能：首页展示未读通知栏，点击查看详情并确认，个人中心展示已确认通知列表，通知详情弹窗采用与协议相同的三列标准布局
- 用户行为轨迹回溯系统：完整应用内实现，路由 `/app/traceback`，包含6个核心页面：仪表盘（统计卡片+带Y轴刻度的柱状图+tooltip）、轨迹列表（搜索/筛选/详情抽屉+生成视频回放预览+可信时间戳®️认证申请弹窗）、轨迹配置（PC/移动端配置表格+新增/编辑/节点配置弹窗）、API集成（系统对接配置+同步日志）、公证申请（批量选择轨迹+申请公证书弹窗）、系统设置（7个Tab的用户管理）；共享Sidebar组件使用useNavigate进行SPA路由跳转；所有表单元素有明确的文字颜色和背景色
- 更新日志管理：位于管理员后台(/admin)，记录真实开发变更，默认数据包含今日(2026-07-06)所有修改；日期格式为中文（如"2026年7月6日"）
- **试用账号云端化**：试用账号存储在Meoo Cloud云端数据库(trial_accounts表)，管理员创建/删除/停用/延期操作直接写入数据库，登录时从云端验证，支持跨用户共享

## Patterns / Constraints
- **访问流程**：用户必须先登录演示平台（管理员或试用账户）→ 选择平台 → 才能使用具体功能
- 可信时间戳®️申请流程：编辑器右侧点击"创作轨迹认证"按钮 → 填写申请表单（文件名称、权利人信息、权利声明）→ 等待5秒认证 → 显示成功并引导查看证书
- 证书展示：包含证书编号、内容哈希、认证时间、有效期等关键信息，PDF证书URL已集成
- 经销商推广banner：首页展示，点击跳转至 https://bd.tsa.cn/dist/evd?code=315728
- 演示账号：demo@example.com / 123456，登录后自动跳转到 /app/scripts
- 演示平台管理员：liugang@tsa.cn / admin@
- 演示平台路由：`/` 登录页 → `/platform` 平台选择（含8个AI演示平台，仅剧本平台可进入）→ `/admin` 管理员后台（需管理员权限，独立根路由避免闪白屏）→ `/app/*` 剧本创作平台
- LOGO统一：登录页(/)和平台选择页(/platform)使用TSA官方LOGO图片（联合信任时间戳服务中心圆形徽标），通过PlatformLogo组件复用
- 编辑器三栏布局：左侧创作阶段导航（含认证标识）、中间内容编辑区、右侧操作轨迹记录（可展开查看）
- 创作阶段：5个阶段(构思/大纲/初稿/修改/定稿)，已认证阶段显示绿色"已认证"徽章
- 操作轨迹：真实记录手动编辑、AI生成、可信时间戳®️认证，存储于 localStorage，仅申请后才显示已认证标识
- AI辅助生成：点击按钮输入提示词，模拟生成剧本内容并记录到轨迹
- 完成阶段流程：点击"完成该阶段" → 弹出价值说明弹窗引导申请认证 → 用户可选择"仅保存"或"完成并申请认证"
- 剧本管理：支持删除用户创建的剧本（预设剧本不可删），数据持久化至 localStorage
- 预设数据：系统预置3个完整剧本（都市情感剧约2000字17条轨迹、悬疑推理约1800字16条轨迹、科幻冒险约2200字23条轨迹），每次加载强制刷新预设内容
- 编辑器右侧按钮：使用"创作轨迹认证"文字，紫色渐变背景醒目展示
- 证书管理：按剧本分组展示，区分阶段证书和过程证书，支持预览PDF和查看原文
- 认证机构：联合信任时间戳服务中心
- 全局术语：统一使用"可信时间戳®️"替代"时间戳"
- 帮助中心FAQ：基于tsa.cn官方话术，包含可信时间戳定义、申请渠道、司法认可、认证周期等核心问题
- 首页定位：AI剧本创作平台，版权保护作为功能模块之一
- 首页风格：参考storyplay.cn深色主题设计，纯黑背景+彩色荧光标题（黄/紫/粉）+手绘下划线SVG效果
- 首页结构：Hero区含海报卡片网格展示、功能特性暗色卡片网格、可信时间戳左右布局评估面板
- 视觉特征：黑色背景(#000)、彩色荧光标题配手绘波浪下划线、深灰卡片(border-gray-800)、白色圆角CTA按钮
- 海报展示：Hero区两行5列网格，使用影视风格真实图片，部分剧本带紫色"已认证"时间戳标识
- Hero标题：主标题"让AI辅助故事创作者成为超级个体"单行显示(whitespace-nowrap)，响应式字号text-3xl/md:text-5xl/lg:text-6xl
- 证书列表：表格形式展示，列包括证书编号、类型、阶段、认证时间、操作（预览/下载/查看原文/验证）
- 证书预览：可拖动弹窗，中英文证书并排显示
- 查看原文：弹窗内预览剧本原文或创作过程记录
- 时间戳申请表单：文件名称自动预设（剧本名-阶段名），权利人类型可选个人/企业，个人需填身份证号，企业需填统一信用代码，权利声明动态显示权利人名称
- 认证等待动画：5秒旋转盾牌动画，提示"正在认证中..."
- 认证成功展示：证书编号、文件名称、权利人、认证时间、有效期（10年），提供"查看证书"快捷入口
